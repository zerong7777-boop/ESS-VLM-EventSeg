# ESS + VLM 事件相机语义分割答辩版说明

## 1. 课题做了什么

本课题研究的是：

`基于视觉语言模型的事件相机语义分割方法`

基础框架采用 ESS。ESS 是事件相机语义分割方向的已有方法，可以在 DDD17 数据集上完成事件流到语义分割结果的训练和验证。

本课题的核心工作不是从零重写一个分割网络，而是在可复现的 ESS 框架上引入视觉语言模型提供的语义先验，观察 DDD17 事件语义分割指标是否提升。

## 2. 为什么要引入视觉语言模型

事件相机数据没有普通 RGB 图像那么直观，标注和语义信息都比较稀缺。视觉语言模型，例如 FC-CLIP，已经在大规模图文数据上学习到了较强的开放语义理解能力。

因此，本课题采用一个工程上更稳妥的路线：

1. 用 FC-CLIP 对与事件数据配对的图像生成离线语义伪标签；
2. 把这些伪标签接入 ESS 的事件分支训练；
3. 再进一步利用教师模型的 top-2 概率信息做 soft distillation。

这种方式的优点是训练时不需要在线调用大模型，显存和时间成本可控，也便于在单卡环境下完成实验。

## 3. 完成了哪些实验

### 3.1 ESS 官方 baseline

首先复现官方 ESS 在 DDD17 上的验证结果：

| 方法 | mIoU | Acc |
| --- | ---: | ---: |
| ESS 官方 baseline | 53.05 | 87.01 |

这个结果作为后续所有 VLM 增强实验的对照基线。

### 3.2 ESS + FC-CLIP dense 伪标签

然后使用 FC-CLIP 生成 dense pseudolabel，并加入事件分支训练。

| 方法 | mIoU | Acc |
| --- | ---: | ---: |
| ESS + FC-CLIP dense 伪标签 | 56.36 | 89.82 |

相对官方 baseline：

- mIoU 提升 `+3.31`
- Acc 提升 `+2.81`

这说明视觉语言模型提供的离线语义伪标签确实能提升事件语义分割效果。

### 3.3 VPR 探索实验

本课题还尝试过基于类别可靠性的 VPR 加权伪标签监督。

| 方法 | mIoU | Acc |
| --- | ---: | ---: |
| VPR 探索版本 | 54.45 | 89.60 |

该结果高于官方 baseline，但没有超过 `ESS + FC-CLIP` 主线，因此不作为最终主推方法，而是作为探索性消融保留。

### 3.4 Top-2 Soft 蒸馏

最终进一步加入 Top-2 Soft 蒸馏。该方法不只使用 FC-CLIP 的 top-1 hard label，还保留教师模型对 top-2 类别的概率信息。

本版最优配置为：

```yaml
weight_top2_soft_distillation: 0.015
top2_soft_temperature: 1.0
top2_soft_min_p1: 0.0
top2_soft_min_margin: 0.0
```

最优结果：

| 方法 | checkpoint | mIoU | Acc |
| --- | --- | ---: | ---: |
| ESS + FC-CLIP + Top-2 Soft 蒸馏 | Epoch 15 | 56.8124 | 89.5776 |

相对官方 baseline：

- mIoU 提升 `+3.7624`
- Acc 提升 `+2.5676`

相对上一版 top-2 soft 最强 checkpoint `56.7029`：

- mIoU 继续提升 `+0.1095`

## 4. 最终推荐结论

最终推荐汇报路线为：

`ESS baseline -> FC-CLIP dense 伪标签 -> Top-2 Soft 蒸馏`

推荐对外汇报的主结果是：

| 方法 | mIoU | Acc |
| --- | ---: | ---: |
| ESS 官方 baseline | 53.05 | 87.01 |
| ESS + FC-CLIP dense 伪标签 | 56.36 | 89.82 |
| ESS + FC-CLIP + Top-2 Soft 蒸馏 best checkpoint | 56.8124 | 89.5776 |

需要说明的是，Top-2 Soft 蒸馏的最优点出现在 `Epoch 15`，最终 `Epoch 18` 为 `56.1059 / 89.6507`。因此该方法的优势应按 best checkpoint 口径表述。

## 5. 答辩时怎么讲创新点

可以从三个层次讲：

第一，完成了 ESS 官方基线在 DDD17 上的复现，建立了可靠实验基础。

第二，将 FC-CLIP 视觉语言模型作为离线语义教师，为事件分支生成 dense pseudolabel，使 ESS 在 DDD17 上从 `53.05` 提升到 `56.36`。

第三，在 hard pseudolabel 的基础上进一步提出 Top-2 Soft 蒸馏模块，利用教师模型的 top-1/top-2 概率分布缓解 hard label 过强监督的问题，最终 best checkpoint 达到 `56.8124`。

## 6. 一句话总结

本课题证明了：在 DDD17 事件相机语义分割任务中，引入 FC-CLIP 视觉语言模型生成的离线语义监督，并结合 Top-2 Soft 蒸馏，可以在 ESS 官方 baseline 上取得明确的 mIoU 提升。
