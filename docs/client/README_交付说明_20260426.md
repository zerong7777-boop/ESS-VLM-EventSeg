# ESS + VLM 项目交付说明（2026-04-26）

## 交付目标

本交付包用于交付“基于视觉语言模型的事件相机语义分割方法研究”的代码、配置、关键权重、推荐 checkpoint 和正式中文说明文档。

本版交付不包含 DDD17 / Cityscapes 原始数据集，也不包含大规模中间缓存和迭代过程文档。

## 当前推荐结果

当前推荐方法为：

`ESS + FC-CLIP 离线伪标签监督 + Top-2 Soft 蒸馏`

主要指标如下：

| 方法 | mIoU | Acc | 说明 |
| --- | ---: | ---: | --- |
| ESS 官方 baseline | 53.05 | 87.01 | 官方 DDD17 UDA 权重验证结果 |
| ESS + FC-CLIP dense 伪标签 | 56.36 | 89.82 | 稳定主线结果 |
| ESS + FC-CLIP + Top-2 Soft 蒸馏 | 56.8124 | 89.5776 | 本版 best checkpoint，Epoch 15 |

相对官方 baseline，最终推荐 best checkpoint 的 mIoU 提升为 `+3.7624`。

相对上一版 top-2 soft 最强结果 `56.7029`，本版 best checkpoint 的 mIoU 提升为 `+0.1095`。

## 包内目录

解压后重点目录如下：

- `project/`：ESS 项目代码、配置、训练脚本和正式文档。
- `weights/official/`：ESS 官方 DDD17 权重。
- `third_party/fc-clip/checkpoints/`：FC-CLIP 与 OpenCLIP 关键权重。
- `artifacts/checkpoints/top2_soft_best_w0015/`：本轮推荐 checkpoint、TensorBoard event 和启动日志。
- `docs/client/`：中文交付说明、方法说明和答辩版傻瓜式文档。

## 使用前提

如需复现实验，需要另行准备：

- DDD17 数据集：放到 `data/ddd17_seg/data`
- Cityscapes 数据集：放到 `data/cityscapes`
- 如需直接训练 FC-CLIP 伪标签版本，还需要准备或重新生成 `data/ddd17_pseudolabels`

如果只查看本版结果和方法实现，直接阅读 `docs/client/` 下文档即可。
